## 01- Mustache 语法
    插值操作
```
    <h2>{{message}}</h2>
```
## v-once 指令
只引用一次
```
        <!-- 数据绑定后，数据变化即更新 -->
        <h2>{{message}}</h2>
        <!-- 只更新一次数据，以后再也不更新 -->
        <h2 v-once>{{message}}</h2>
```
## v-html 指令
将变量或值翻译成标准的html格式
```
        <!-- v-html 将变量解析为标准的html格式语法 -->
        <h2 v-html ="url"></h2>
```
